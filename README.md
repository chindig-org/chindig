
## Chindig

Discover the Indie Chicago Community.

## Requirements

- node (8.9.1+)
- npm (5.6.0+)

## Local Installation

```
cd chindig
npm install
```

## Development

Serve and listen for changes with

```
npm start
```

## Build

Creates optimized production build in `build/`

```
npm run build
```
